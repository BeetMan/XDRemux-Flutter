约定：

- 正常转换入口应优先使用 `xdremux/` 下的当前 CLI，而不是这里的旧实验快照。
- 每个实验目录应说明假设、输入条件、验证结果和是否已经被主线吸收。
- 不要从实验目录静默复制代码进入生产路径；需要通过 PR 同时更新生产实现、测试和文档。
- 大体积 HEIC、APK、样本包不要直接提交到 git。实验记录应保留文件 hash、来源说明和复现实验所需的最小步骤。
